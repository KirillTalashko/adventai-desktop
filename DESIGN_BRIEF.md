# Мета-промпт: редизайн AdventAI Desktop в стиле Claude Code

> Это бриф-инструкция для самого себя (Claude) при переделке UI. Цель — не копировать Android-приложение,
> а воспроизвести **ощущение Claude Code**: спокойный минимализм, воздух, плавность, всё на своём месте.

## Принципы

1. **Тихо и просторно.** Светлый, почти белый фон. Минимум линий и теней. Крупные отступы, дыхание.
2. **Один акцент.** Нейтральная серо-чёрная база + один акцент (синий AdventAI) только для активного/важного.
3. **Скругления везде.** Радиусы 10–16. Кнопки-«пилюли», карточки, поле ввода — скруглённые.
4. **Линейные иконки** (Material Outlined), не цветные. Иконка + короткий текст.
5. **Состояния плавные.** Hover/active подсветка диалогов, спиннер при ответе, автоскролл.

## Раскладка

```
┌───────────────┬───────────────────────────────────────────────┐
│  SIDEBAR ~272  │  ВЕРХ: тонкий заголовок диалога                 │
│                │                                                 │
│  ✦ AdventAI    │  ЛЕНТА сообщений (воздух, без тяжёлых рамок):    │
│  Визовый спец. │   • пользователь — мягкая «пилюля» справа        │
│                │   • агент — текст слева + чип токенов снизу      │
│ [+ Новая       │   • чек-лист документов — карточка со статусами  │
│    сессия]     │                                                 │
│                │                                                 │
│  Диалоги       │  ── пусто: подсказка + чипы-примеры ──           │
│  • Диалог 1    │                                                 │
│  • Диалог 2 ✕  │  НИЗ — КОМПОЗЕР (главный элемент):               │
│  ...           │  ┌───────────────────────────────────────────┐ │
│                │  │ поле ввода «Спросите визового специалиста…»│ │
│ [Память][⚙]    │  │ [≡ модель ▾] [🧠 память ▾]  ⚡toks   [ ↑ ]  │ │
│                │  └───────────────────────────────────────────┘ │
└───────────────┴───────────────────────────────────────────────┘
```

- **Слева:** логотип-марк, кнопка **«+ Новая сессия»** (заметная), секция «Диалоги» со списком; активный — с тонкой подсветкой/левой акцентной полосой; ✕ на наведении. Внизу — «Память» и «Настройки».
- **Снизу (как в Claude Code):** скруглённый композер. Под полем — панель: **выбор модели** (dropdown), **выбор режима памяти** (dropdown), **счётчик токенов** сессии, справа круглая **кнопка отправки** (стрелка вверх, тёмная). Отправка по Enter, Shift+Enter — перенос.

## Палитра

- bg `#FAFAF9`, surface `#FFFFFF`, sidebar `#F4F4F2`, border `#E7E7E3`
- текст `#1C1C1A`, приглушённый `#6B6B66`
- primary (кнопка отправки/действия) — тёмный `#1A1A19`, иконка белая
- accent (активное, ссылки) — синий `#2F6BED`
- статусы документов: нужен `#6B7280` · загружен `#2F6BED` · проверен `#2E7D32` · не хватает `#B26A00`

## Память (режимы) — селектор в композере

- **3 слоя** (Day 11): долговременная (профиль/решения) + рабочая (задача) + последние N. Инъекция блоками.
- **Sliding window**: только последние N сообщений.
- **Key-value (факты)**: извлекаем key-value факты из старого хвоста (LLM) и подмешиваем блоком.
- **Summary**: сворачиваем старую часть в краткое резюме (LLM) и подмешиваем.
- **Полная история**: всё как есть.

Производные (summary/facts) хранятся в самом диалоге и обновляются, когда «старый хвост» вырос.

## Токены

- Под каждым ответом агента — чип `↑prompt ↓completion · Σtotal`.
- В композере — суммарные токены сессии (и стоимость, если у модели есть тариф).

## Иконки (Material Outlined)

Add (новая сессия), ArrowUpward (отправка), KeyboardArrowDown (dropdown), Settings, DeleteOutline (удалить диалог), Bolt/токены. Логотип — скруглённый квадрат с «A».
